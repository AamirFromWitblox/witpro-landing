## macOS Installation Guide

### Step 1: Download the latest version

1. Download the latest dmg file from [here.](/download?platform=mac)
2. Once downloaded, open the DMG file.

### Step 2: Install WitPro

1. Drag the WitPro icon to the `Applications` folder.

### Step 3: Bypass Gatekeeper

Upon trying to open WitPro, you may encounter a message indicating the "wit-pro" can't be opened.

1.  Open settings.
