## Windows Installation Guide

### Step 1: Download the latest version

- Download the latest exe file from [here.](/download?platform=windows)

### Step 2: Install WitPro

- Once downloaded, double click the EXE file.

### Step 3: Bypass Gatekeeper

Upon trying to open WitPro, you may encounter this warning.

![enter image description here](https://firebasestorage.googleapis.com/v0/b/witpro-e38b9.appspot.com/o/docs%2Fsmartscreen-1.png?alt=media&token=97f3eee3-09cd-4537-9da4-8467166766c9)

To bypass this warning:
`Click on More Info -> Run Anyway.`

You should now have a fully functional WitPro on your Windows.
