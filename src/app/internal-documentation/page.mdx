### 1. **digital_write**

- **Name**: digital_write
- **Description**: This block is used to set a digital pin to a specified state (HIGH or LOW).
- **Required Fields**:
  - `pinType`: Specifies the state to set the digital pin to, either HIGH or LOW.
    - **Type**: `String`
    - **Options**: "HIGH", "LOW"
- **Attachments**:
  - `pinNumber`: Input block to specify the pin number. It can be a `get_variable` block or a `math_number` block.

### 2. **digital_read**

- **Name**: digital_read
- **Description**: This block reads the state of a digital pin (HIGH or LOW).
- **Required Fields**: None.
- **Attachments**:
  - `pinNumber`: Input block to specify the pin number to read from. It can be a `get_variable` block or a `math_number` block.

### 3. **analog_write**

- **Name**: analog_write
- **Description**: This block is used to write an analog value (PWM wave) to a specific pin.
- **Required Fields**: None.
- **Attachments**:
  - `pinNumber`: Input block to specify the pin number. It can be a `get_variable` block or a `math_number` block.
  - `intensity`: Input block to specify the intensity (a value between 0-255). It can be a `get_variable` block, `math_number` block, or other arithmetic blocks.

### 4. **analog_read**

- **Name**: analog_read
- **Description**: This block reads the value from an analog pin (a value between 0-1023).
- **Required Fields**: None.
- **Attachments**:
  - `pinNumber`: Input block to specify the analog pin number to read from. It can be a `get_variable` block or a `math_number` block.

### 5. **pinmode**

- **Name**: pinmode
- **Description**: This block sets a specified pin to either INPUT or OUTPUT mode.
- **Required Fields**:
  - `pinType`: Specifies the mode of the pin, either INPUT or OUTPUT.
    - **Type**: `String`
    - **Options**: "INPUT", "OUTPUT"
- **Attachments**:
  - `variable`: Input block to specify the pin number. It can be a `get_variable` block or a `math_number` block.

### 6. **delay**

- **Name**: delay
- **Description**: This block pauses the program for a specified duration (in milliseconds).
- **Required Fields**:

- **Attachments**: None.

### 7. **setup_and_loop**

- **Name**: setup_and_loop
- **Description**: This block defines the `setup()` and `loop()` functions of an Arduino sketch. The `setup()` function is run once when the program starts, and the `loop()` function is run repeatedly after that.
- **Required Fields**: None.
- **Attachments**:
  - `SETUP_CODE`: Input block to specify the code that will run in the `setup()` function.
  - `LOOP_CODE`: Input block to specify the code that will run in the `loop()` function.
